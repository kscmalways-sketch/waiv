    🌐 인터넷에 올리는 방법

    생성된 사이트를 인터넷에 무료로 공개하는 방법입니다. 위의 코드 복사 버튼을 먼저 클릭하세요.

    방법 1 — Netlify (가장 쉬움, 추천 ⭐)
    1. 위의 코드 복사 버튼 클릭
    2. 컴퓨터에서 메모장(Windows) 또는 텍스트 편집기(Mac) 열기
    3. 붙여넣기(Ctrl+V 또는 Cmd+V) 후 파일 이름을 index.html로 저장⚠️ Windows 메모장: 저장 시 "파일 형식"을 반드시 "모든 파일(*.*)"로 변경하세요
    4. netlify.com 접속 → Sign up으로 무료 회원가입
    5. 로그인 후 상단 Sites 메뉴 클릭
    6. 페이지 하단 "파일을 여기에 드래그하거나 클릭하여 업로드" 영역에 저장한 index.html을 드래그 앤 드롭
    7. 잠시 후 자동으로 인터넷 주소(예: amazing-site-abc123.netlify.app)가 생성됩니다. 🎉원하면 Site settings에서 주소 이름을 변경할 수 있어요
    방법 2 — GitHub Pages (무료 영구 주소)
    1. github.com 접속 → 무료 회원가입
    2. 로그인 후 우측 상단 + 버튼 → New repository 클릭
    3. Repository name에 원하는 이름 입력 (예: my-website) → Public 선택 → Create repository 클릭
    4. 생성된 페이지에서 "uploading an existing file" 링크 클릭
    5. 저장한 index.html 파일을 드래그 앤 드롭 → 하단 Commit changes 클릭
    6. 상단 Settings 탭 클릭 → 좌측 메뉴에서 Pages 클릭
    7. Source: Deploy from a branch 선택 → Branch: main 선택 → Save 클릭
    8. 2~3분 후 사용자이름.github.io/my-website 주소로 접속 가능! 🎉
